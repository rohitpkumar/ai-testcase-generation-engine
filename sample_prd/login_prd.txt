Product: Web Application Login Module

Requirements:

1. User must be able to login using registered email and password.
2. System should validate email format.
3. Password must be minimum 8 characters.
4. User account should lock after 5 failed attempts.
5. Forgot password option should send reset email.
6. System should display proper error message for invalid login.
7. Login API response time should be less than 2 seconds.
